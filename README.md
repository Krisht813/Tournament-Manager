# Tournament Manager

A modern, beautiful tournament management platform built with Next.js 14, React, and Tailwind CSS.

## Features

- 🏆 **Tournament Management** - Organize and track tournaments with ease
- 🎨 **Beautiful UI** - Modern gradient design with smooth animations
- 🌓 **Dark Mode** - Persistent theme across all pages
- 📱 **Responsive Design** - Works perfectly on all devices
- ⚡ **Fast Performance** - Built with Next.js 14 for optimal speed
- 🔐 **Authentication Pages** - Login and signup with social auth options
- 📄 **Legal Pages** - Privacy policy, terms of service, and cookie settings

## Tech Stack

- **Framework:** Next.js 14
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **UI Components:** Radix UI
- **Animations:** Framer Motion
- **Icons:** Lucide React

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/tournament-manager.git
cd tournament-manager
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Pages

- **Home** - Landing page with features, pricing, testimonials, and FAQ
- **Login** - User authentication page
- **Sign Up** - User registration page
- **Privacy Policy** - Privacy and data protection information
- **Terms of Service** - Terms and conditions
- **Cookie Settings** - Cookie preferences management

# Tournament Manager Backend

Django REST API for Tournament Manager application.

## Setup Instructions

### 1. Create Virtual Environment

```bash
cd backend
python -m venv venv
```

### 2. Activate Virtual Environment

**Windows:**
```bash
venv\Scripts\activate
```

**Mac/Linux:**
```bash
source venv/bin/activate
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Configure Environment Variables

Update the `.env` file with your settings:
- `SECRET_KEY`: Django secret key
- `EMAIL_HOST_USER`: Your Gmail address
- `EMAIL_HOST_PASSWORD`: Your Gmail app password

### 5. Run Migrations

```bash
python manage.py makemigrations
python manage.py migrate
```

### 6. Create Superuser (Optional)

```bash
python manage.py createsuperuser
```

### 7. Run Development Server

```bash
python manage.py runserver
```

The API will be available at `http://localhost:8000`

## API Endpoints

### Request Password Reset
```
POST /api/accounts/request-password-reset/
Body: { "email": "user@example.com" }
```

### Reset Password
```
POST /api/accounts/reset-password/
Body: { "token": "uuid-token", "password": "newpassword123" }
```

## Testing Email

You can test the email functionality by:
1. Creating a user in Django admin
2. Sending a POST request to `/api/accounts/request-password-reset/`
3. Checking your email for the reset link

## Production Deployment

For production:
1. Set `DEBUG=False` in `.env`
2. Update `ALLOWED_HOSTS` with your domain
3. Use a production database (PostgreSQL recommended)
4. Set a strong `SECRET_KEY`
5. Configure proper email settings
